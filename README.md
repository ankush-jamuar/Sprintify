# 🚀 Sprintify

**AI-Powered Project Management SaaS Platform**

Sprintify is a full-stack, production-ready SaaS application designed to streamline team collaboration, sprint planning, and task management with AI-driven insights.

🌐 **Live Demo:** https://sprintify-woad.vercel.app

---

## ✨ Key Features

* 🔐 Secure JWT-based Authentication
* 🏢 Workspace & Team Collaboration
* 📁 Project, Task, and Sprint Management
* 📊 Drag-and-Drop Kanban Board
* 🧠 AI Copilot (task generation, summaries, insights)
* 🔔 Real-time Notifications & Activity Tracking
* ⚡ Real-time updates using Socket.io

---

## 🛠 Tech Stack

**Frontend:** React (Vite + TypeScript), TailwindCSS, React Query, Framer Motion
**Backend:** Node.js, Express, MongoDB (Mongoose), Socket.io
**AI:** Groq API
**Deployment:** Vercel (Frontend), Render (Backend), MongoDB Atlas

## ⚙️ Environment Variables

### Backend

```

MONGO_URI=<your_mongodb_uri>
JWT_SECRET=<your_secret>
CLIENT_URL=<your_frontend_url>
GROQ_API_KEY=<your_key>
```

### Frontend

```
VITE_API_BASE_URL=<your_backend_api_url>
VITE_SOCKET_URL=<your_backend_socket_url>
```

## 🚀 Getting Started

```bash
git clone https://github.com/ankush-jamuar/Sprintify.git
cd Sprintify
```

```bash
# Backend
cd backend
npm install
npm run dev
```

```bash
# Frontend
cd frontend
npm install
npm run dev
```
